# Lead Quality Audit Playbook

## Purpose

Use this when reported lead volume improves but the business outcome does not.

The goal is to understand whether the system is creating qualified demand or simply creating more records.

## Questions to answer

- Which sources produce leads that progress?
- Which sources create volume without movement?
- Which records are missing source detail?
- Which forms, calls, or paths produce stronger downstream outcomes?
- Which conversion is the platform optimizing toward?
- Which records should be excluded from optimization?

## Audit structure

### 1. Define the funnel

Create consistent definitions for:

- Lead
- Qualified lead
- Connected lead
- Application or opportunity
- Enrollment, sale, or final outcome

### 2. Pull source-level movement

Review each source by:

- Volume
- Cost
- Connect rate
- Qualification rate
- Progression rate
- Final outcome rate

### 3. Identify record-quality issues

Look for:

- Missing source
- Missing campaign
- Duplicate records
- Conflicting form values
- Inconsistent grade, product, market, or segment fields
- Call records without outcome labels

### 4. Separate acquisition issue from process issue

Not every weak lead is a media problem.

It may be:

- A handoff issue
- A routing issue
- A follow-up timing issue
- A form-quality issue
- A message mismatch
- A source-quality issue

### 5. Recommend action

Possible actions:

- Shift investment
- Exclude low-quality signals
- Improve routing
- Add required fields
- Remove unnecessary friction
- Improve source capture
- Adjust optimization event
- Add nurture for slower-converting leads

## Output

A useful audit should produce:

- Source-quality table
- Record-quality issue log
- Recommended investment changes
- Measurement cleanup list
- Follow-up workflow improvements
- Executive summary

## Standard

Do not call a lead source efficient until it shows downstream quality.
