# Brand and Voice

This section defines how Jared's professional narrative should sound and how it should be translated across public-facing artifacts.

The goal is consistency without making the work feel over-polished or generic.

## Files

- `personal-brand-system.md` — positioning, audience, proof posture, and brand themes
- `writing-style-guide.md` — tone, rules, voice markers, and editing standards

## Core positioning

Jared stabilizes and scales growth systems.

The strongest brand frame is not narrow paid media execution. It is growth architecture: the operating layer behind performance media, CRO, analytics, agency governance, executive communication, and AI-enabled workflows.

## Voice standard

The voice should be:

- clear
- direct
- warm
- precise
- practical
- understated
- executive-safe

It should avoid inflated language, generic thought leadership, unsupported certainty, and anything that sounds like it was written to impress instead of clarify.

## How to use this section

Use this section before creating:

- executive bios
- recruiter summaries
- website copy
- LinkedIn positioning
- case-study intros
- interview talking points
- public-facing repo language

## Standard

Clear beats clever. Useful beats polished. Specific beats impressive.
