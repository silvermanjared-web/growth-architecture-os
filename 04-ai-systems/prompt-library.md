# Prompt Library

These prompts are written for practical growth, media, CRO, and executive communication work.

## Executive brief prompt

```txt
Turn the notes below into an executive-ready brief.

Audience: [audience]
Decision needed: [decision]
Tone: clear, direct, calm, senior

Structure:
1. Bottom line
2. What changed
3. Why it matters
4. Recommendation
5. Watchouts
6. Next steps

Notes:
[paste notes]
```

## Performance readout prompt

```txt
Analyze this performance update and convert it into a decision-ready readout.

Separate:
- What we know
- What we suspect
- What we cannot trust yet
- What action is recommended

Avoid channel commentary unless it changes the business decision.

Data / notes:
[paste]
```

## CRO roadmap prompt

```txt
Build a CRO roadmap from the audit notes below.

Use this structure:
- Situation
- Diagnosis
- Thesis
- Roadmap at a glance
- Phase 0: foundation
- Phase 1: credibility and reassurance
- Phase 2: structural friction
- Phase 3: capture and scale
- RACI
- Timeline
- Scorecard
- This-week actions

Make the sequence dependency-based, not idea-based.

Audit notes:
[paste]
```

## Political QA prompt

```txt
Review this draft for executive and stakeholder risk.

- Tone that may create defensiveness
- Overstatement
- Ambiguous ownership
- Missing decision or ask
- Too much detail
- Anything that sounds reactive or prosecutorial

Then rewrite it in a clear, senior, direct tone.

[paste]
```
