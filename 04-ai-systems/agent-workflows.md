# Agent Workflows

## Purpose

These are practical AI-assisted workflows for growth and marketing operations.

They are intentionally narrow. The goal is to create useful leverage without turning every task into an overbuilt automation project.

## Workflow 1: Weekly performance brief

### Input

- Channel performance export
- Funnel movement
- Budget pacing
- Known anomalies
- Business goal or forecast

### Agent task

Create a leadership-ready weekly brief with:

- Bottom line
- What changed
- Why it matters
- Risks
- Recommended actions
- Open questions

### Human review

Check:

- Are claims supported?
- Are caveats clear?
- Is the recommendation practical?
- Does the tone avoid unnecessary alarm?

## Workflow 2: CRO roadmap conversion

### Input

- Audit notes
- Page inventory
- Funnel data
- Heatmap or session notes
- Stakeholder constraints

### Agent task

Convert findings into:

- Situation
- Diagnosis
- Thesis
- Prioritized roadmap
- RACI
- Timeline
- Scorecard
- This-week actions

### Human review

Check:

- Is the sequence dependency-based?
- Are metrics clear?
- Are owners realistic?
- Are public claims compliant and accurate?

## Workflow 3: Agency readout QA

### Input

- Agency weekly update
- Internal performance data
- Prior commitments
- Open issues

### Agent task

Summarize:

- What the agency said
- What the data supports
- What is missing
- What needs a decision
- Follow-up questions

### Human review

Check:

- Is the critique fair?
- Is the ask specific?
- Is accountability clear without creating unnecessary defensiveness?

## Workflow 4: Executive communication polish

### Input

- Draft email, memo, or update
- Audience
- Objective
- Desired decision or next step

### Agent task

Revise for:

- Clarity
- Brevity
- Tone
- Ownership optics
- Political risk
- Decision-readiness

### Human review

Check:

- Does it sound like Jared?
- Is it too long?
- Is it too intense?
- Is the ask clear?

## Workflow 5: Playbook extraction

### Input

- Completed project
- Deck
- Memo
- Audit
- Readout

### Agent task

Extract the reusable method:

- When to use it
- Inputs
- Steps
- Outputs
- Risks
- Template

### Human review

Check:

- Is it genuinely reusable?
- Is anything confidential?
- Does the playbook preserve judgment, not just steps?

## Standard

Good agent workflows reduce repeated cognitive load while preserving human accountability.
